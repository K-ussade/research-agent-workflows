# Contributing

## Purpose

Contributions should improve public-safe research workflow documentation, citation discipline, source review, editorial gates, and AI assistance disclosure.

## Contribution Rules

| Area | Rule |
| --- | --- |
| Examples | Use synthetic examples only. |
| Sources | Reference public sources by citation; do not copy scraped copyrighted content. |
| Editorial authority | Preserve human editorial review before publication or public claim use. |
| Sensitive data | Do not submit private research, private prompts, model outputs, donor data, student data, volunteer data, customer data, secrets, or sealed IP. |
| Claims | Do not add unsupported publication outcomes, client outcomes, impact claims, or active service claims. |
| Graphs | Update diagrams when workflow, source, review, publishing, or boundary behavior changes. |

## Pull Request Checklist

- [ ] The change is public-safe.
- [ ] Examples are synthetic.
- [ ] Source review and citation expectations are explicit.
- [ ] No scraped copyrighted content or copied paywalled text is included.
- [ ] No private research notes, private prompts, model outputs, credentials, private data, or sealed IP are included.
- [ ] Human editorial review remains required before publication.
- [ ] `scripts/validate-research-agent-workflows.sh` passes.

## Security And Privacy

Do not open public issues containing private research, private documents, sensitive source material, credentials, tokens, private prompts, private model outputs, private Foundation operations, or sealed YOSO-YAi LLC IP.
