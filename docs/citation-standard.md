# Citation Standard

## Purpose

This standard defines citation tracking for AI-assisted research briefs.

## Citation Log Fields

| Field | Description |
| --- | --- |
| Citation ID | Stable identifier such as `SRC-001`. |
| Source title | Public title or formal citation. |
| URL | Public link when available. |
| Publisher or author | Source owner or author when known. |
| Date accessed | `YYYY-MM-DD`. |
| Source type | Primary, official, academic, news, documentation, dataset, or other. |
| Supported claim | Specific claim or context supported. |
| Quote status | None, short reviewed quote, or paraphrase only. |
| Review status | Draft / review-required, approved, limited, or rejected. |

## Citation Rules

- Every factual claim in a public research brief should trace to a citation or be removed.
- AI-generated summaries are not citations.
- Do not fabricate citation details.
- Do not use private research notes as public citations.
- Do not copy scraped copyrighted content into the citation log.

## Review Rule

Citation logs require human review before publication. The reviewer checks source relevance, claim fit, status label, and copyright-safe handling.
