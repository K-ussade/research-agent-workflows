# Research Agent Workflows

## Purpose

`research-agent-workflows` is a public-safe template and workflow repository for AI-assisted research operations, source review, citation tracking, research briefs, publishing pipelines, and editorial review.

It supports documented research and agentic publishing workflows without exposing private research notes, private prompts, unpublished sensitive content, private Foundation operations, scraped copyrighted content, or unreviewed AI output.

This repository is documentation-only. It does not run autonomous research agents, scrape sources, publish automatically, bypass editorial review, or claim that AI output is authoritative.

## Public Boundary

Allowed public material:

- public-safe workflow standards;
- citation and source-review templates;
- editorial review gates;
- AI assistance disclosure language;
- synthetic examples;
- diagrams that show process, authority, and publication boundaries.

Forbidden material:

- private research notes, private prompts, private model outputs, private Foundation operations, customer data, donor data, student data, volunteer private data, or sealed YOSO-YAi LLC IP;
- scraped copyrighted content, copied paywalled material, unpublished sensitive content, or private documents;
- credentials, tokens, API keys, secrets, account screenshots, private training corpora, or exact sensitive infrastructure locations;
- security-sensitive NEURONA operational details;
- unsupported claims, client outcomes, publication outcomes, or active service claims.
