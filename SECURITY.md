# Security Policy

## Reporting Sensitive Issues

Do not open a public issue containing credentials, tokens, API keys, secrets, private research notes, private prompts, private model outputs, donor data, student data, volunteer private data, customer data, private Foundation operations, exact sensitive infrastructure locations, security-sensitive NEURONA operational details, or sealed YOSO-YAi LLC IP.

Use a private maintainer contact path until a formal security process is published.

## Public Issue Scope

Public issues may discuss:

- template clarity;
- broken links;
- citation field improvements;
- graph rendering;
- public boundary language;
- validation failures that do not expose sensitive content.

Public issues must not include:

- copied paywalled text or scraped copyrighted content;
- unpublished sensitive content;
- private research notes;
- private prompts or model outputs;
- private operations records;
- credentials, tokens, API keys, or secrets;
- private training corpora;
- security-sensitive infrastructure details.

## Supported Scope

This repository is documentation-only. It does not scrape websites, collect data, run agents, publish articles, host models, publish datasets, or operate demos.
